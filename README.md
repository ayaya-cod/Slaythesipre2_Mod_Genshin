# 🎮 Genshin Posion Mod for Slay the Spire 2
一款为《杀戮尖塔2》注入原神元素的自定义模组，带来全新的卡牌、遗物、药水等内容，让你在尖塔中体验原神的战斗乐趣！

---

## ✨ 模组内容
- 全新自定义卡牌：融入原神角色技能、元素反应机制
- 特色遗物：还原原神圣遗物、道具效果，打造专属build
- 定制药水：新增原神主题药水，丰富战斗策略
- 持续更新：后续将加入更多角色、卡牌与玩法内容

---

## 📦 安装教程
### 1. 准备工作
- 确保你已安装《杀戮尖塔2》游戏本体
- 首次安装模组需在**游戏根目录**新建文件夹，命名为 `mods`

### 2. 安装步骤
1.  从 Release 下载最新版模组压缩包
2.  解压后得到 `Genshin_mod` 文件夹（模组本体）
3.  将 `Genshin_mod` 整个文件夹放入游戏根目录的 `mods` 文件夹中
4.  启动游戏，即可体验模组内容
